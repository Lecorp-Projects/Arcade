Guía Maestra: Lógica y Sintaxis de Python (Nivel Base)

Este documento contiene las reglas de construcción con las que armaremos el arcade.py y sus módulos.
1. Estructuras de Control y Flujo

Son los "rieles" por donde corre tu código.

    while <condición>:: Se usa para el bucle principal del juego. Mientras la condición sea verdadera, el juego no termina.

    if / elif / else: Para la toma de decisiones (ej: ¿el usuario eligió el juego 1 o el 2?).

    for <variable> in <secuencia>:: Esencial para recorrer tus matrices (el tablero de 4 en línea) o listas de palabras.

    break / continue:

        break: Rompe el ciclo inmediatamente (útil cuando alguien gana).

        continue: Salta a la siguiente iteración del ciclo.

2. Definición y Alcance (Funciones)

    def nombre_funcion(parámetros):: Bloques de construcción.

    return <valor>: Envía un resultado de vuelta. Si no pones return, la función devuelve None.

    global <variable>: Úsalo con cuidado. Permite que una función modifique una variable que está afuera de ella (como el puntaje total del Arcade).

3. Manejo de Listas (Tu "Memoria" manual)

Como quieres evitar los métodos automáticos, estos son tus operadores base:

    Acceso por índice: lista[i] (Leer) o lista[i] = valor (Escribir).

    Troceado (Slicing): lista[inicio:fin] (Crea una copia de una parte de la lista).

    len(lista): Te da el tamaño actual. Fundamental para tus límites de ciclos range().

    Concatenación: lista_final = lista1 + lista2 (Crea una nueva lista uniendo ambas).

4. Importaciones Modularizadas

Para que tu proyecto esté organizado, usaremos esta sintaxis:

    En el archivo main.py:
    Python

    import ahorcado  # Importa todo el archivo
    from ppt import jugar_ppt  # Importa solo una función específica

    Llamada: Si usaste import ahorcado, llamas a la función como ahorcado.iniciar_juego().

5. Operadores Lógicos (Para el 4 en línea)

Para verificar victorias complejas:

    and: Ambas condiciones deben ser ciertas.

    or: Al menos una debe ser cierta.

    not: Invierte el valor de verdad.

    in: Verifica si un elemento existe dentro de una colección.

📂 Guía Maestra: Importaciones y Herramientas de Interacción
1. Importar archivos propios (Módulos)

Como vas a separar los juegos en archivos .py diferentes, necesitas saber cómo conectarlos al main.py.

    Importar todo el archivo:
    Python

    import ahorcado
    # Para usar una función de ese archivo:
    ahorcado.jugar()

    Importar una función específica:
    Python

    from ppt import jugar_ppt
    # La usas directamente:
    jugar_ppt()

    Importar con un alias (para escribir menos):
    Python

    import cuatro_en_linea as cel
    cel.iniciar()

    Regla de oro: Todos los archivos deben estar en la misma carpeta para que Python los encuentre fácilmente.

2. Entrada y Salida de Datos (Input / Output)

Es la base de cualquier juego en terminal.

    input("Mensaje: "): Detiene el programa y espera a que el usuario escriba algo. Ojo: Siempre devuelve un texto (string).

    Conversión de tipos (Casting): Como el input es texto, si necesitas un número para elegir una opción del menú, debes convertirlo:
    Python

    opcion = int(input("Elige un juego: "))

    print(): Muestra información.

        print("Hola", nombre): Espacia automáticamente los elementos.

        print(f"Puntaje: {puntos}"): (F-string) La forma más limpia de meter variables en texto.

3. Manejo de Strings (Cadenas de texto)

Esencial para el Ahorcadito.

    string.upper() / string.lower(): Convierte todo a mayúsculas o minúsculas (útil para que "A" y "a" sean lo mismo en el juego).

    list(cadena): Convierte una palabra en una lista de letras. Muy útil si quieres cambiar una letra específica, ya que los strings en Python son inmutables (no se pueden cambiar letra por letra directamente).

4. Manejo de Errores Básico (try / except)

Para que tu arcade.py no se cierre si el usuario escribe una letra cuando pides un número:
Python

try:
    opcion = int(input("Escribe un número: "))
except ValueError:
    print("Eso no es un número válido.")

5. La función range() (Tu mejor amiga para ciclos)

La usaste en tus funciones, pero recuerda sus 3 formas:

    range(stop): De 0 hasta stop - 1.

    range(start, stop): De start hasta stop - 1.

    range(start, stop, step): Igual, pero saltando de tanto en tanto (o hacia atrás con -1).
